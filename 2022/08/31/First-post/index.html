Two Sum

2022/8/31

Leetcode no.1 : Two Sum

一、 题目

Given an array of integers nums and an integer target, return indices of the two numbers such that they add up to target.

You may assume that each input would have exactly one solution, and you may not use the same element twice.

You can return the answer in any order.

Example 1:

Input: nums = [2,7,11,15], target = 9
Output: [0,1]
Explanation: Because nums[0] + nums[1] == 9, we return [0, 1].

Example 2：

Input: nums = [3,2,4], target = 6
Output: [1,2]

二、思路

###显然可以利用两个循环进行嵌套，得出答案。但是此算法时间复杂度为 $O(n^{2})$， 不是最理想的选择。此时，我们应想到利用哈希表，提高算法效率。

###哈希表（HashMap) 在此题中可以做到边搜索边存储，因此只需遍历一遍数列即可找到答案。

具体来说，我们利用STL中的map来实现哈希表。

例如

Input: nums = [2,7,11,15], target = 9
Output: [0,1]

map[2]=1, map[7]=2, map[11]=3 …

如此，我们就将nums中的每一个数都对应到了一个从1开始的顺序值。

随后我们开始遍历，显然，我们每次都将nums[i]加入哈希表map。遍历时每次都要用target-nums[i], 并且在哈希表中搜索这个值，如果有，那么break，return二者序列号即可，可以使用 stl 的 vector 装这两个值。

三、code

class Solution {
public:
vector<int> twoSum(vector<int>& nums, int target) {
map<int,int> m;
vector<int> v;
for(int i=0;i<nums.size();i++){
    int dif = target - nums[i];
    if(m.find(dif) != m.end()){
        v.push_back(m[dif]);
        v.push_back(i);
        break;
    }
    m[nums[i]]=i;
    }

return v;

}
};
